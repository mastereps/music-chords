# AGENTS.md

## Project rules
- Build for mobile first
- This is a chords-only web app, no audio features
- Prefer readability over decorative UI
- Use React functional components and hooks
- Use TypeScript everywhere possible
- Use Tailwind for styling
- Backend must use Express + POSTGRESQL with connection pooling
- Validate all inputs
- Add explicit error handling and logging
- Keep components small and reusable
- Put business logic in utilities/services, not directly in UI
- Preserve chord spacing and line breaks
- Use monospace font for chord blocks
- Add tests for transpose logic
- Do not introduce unnecessary dependencies
